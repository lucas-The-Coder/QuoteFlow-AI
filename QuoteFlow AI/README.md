# QuoteFlow AI - Frontend

React + TypeScript + Tailwind UI for QuoteFlow AI.

## Setup
1. cd quoteflow-ui-improved
2. npm install
3. npm start

Runs on http://localhost:3000
Expects backend at http://localhost:5284/api

## Features
- Gradient auth pages (royalblue → cyan)
- JWT auth with localStorage
- Dashboard with metrics
- Quotes list & create
- Customers management
- Protected routes
- Auto-logout on 401
